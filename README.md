# graphicAI
showcase of OpenGL using WebGL anf Three.js.

start the front-end using following commands:

``````bash
cd client
npm install
npm start
``````

## Download pre-trained model
[PointRend](https://github.com/ayoolaolafenwa/PixelLib/releases/download/0.2.0/pointrend_resnet50.pkl) from here.

Place it in root folder

## Install requirements
```sh
pip install -r requirements.txt
python3 server.py
```
