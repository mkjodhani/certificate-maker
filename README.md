# certificate-maker
Web Application to generate different kind of certificates.

### Dependencies

<a href="https://trufflesuite.com/ganache/">Ganache</a>

start the front-end using following commands:

``````bash
cd client
npm install
npm start
``````

start the back-end using following commands:

``````bash
cd server
npm install
npm start
``````
